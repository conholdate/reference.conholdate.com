## Импорт необходимых пакетов

Откройте основной файл C# и импортируйте необходимые пространства имен:

```csharp
using System.IO;
using Aspose.Pdf;
using System;
```

Давайте разобьем процесс конвертации на управляемые этапы.

## Шаг 1: Определите каталог данных

Укажите путь к каталогу ваших документов:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // Замените фактическим путем к папке «Документы».
```

## Шаг 2: Создайте HtmlLoadOptions

 Далее создайте экземпляр`HtmlLoadOptions` чтобы настроить способ загрузки внешних ресурсов:

```csharp
HtmlLoadOptions options = new HtmlLoadOptions();
options.CustomLoaderOfExternalResources = new LoadOptions.ResourceLoadingStrategy(SamePictureLoader);
```

Такая настройка гарантирует правильную обработку изображений и других ресурсов во время конвертации.

## Шаг 3: Загрузите HTML-документ

 Теперь загрузите ваш HTML-документ в Aspose.PDF.`Document` объект:

```csharp
Document pdfDocument = new Document(dataDir + "HTMLToPDF.html", options);
```

## Шаг 4: Сохраните PDF-документ.

Наконец, сохраните преобразованный документ как файл PDF:

```csharp
pdfDocument.Save("HTMLToPDF_out.pdf");
```

Это сохранит выходной PDF-файл в том же каталоге, что и ваш HTML-файл.

## Шаг 5: Обработка исключений

Оберните свой код в блок try-catch, чтобы избежать возможных ошибок в процессе преобразования:

```csharp
try
{
    // Ваш код преобразования здесь
}
catch (Exception ex)
{
    Console.WriteLine("An error occurred: " + ex.Message);
}
```

Эта практика поможет вам выявить проблемы, которые могут возникнуть в ходе выполнения.
