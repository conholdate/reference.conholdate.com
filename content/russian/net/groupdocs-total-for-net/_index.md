**Welcome to the GroupDocs.Annotation for .NET**

GroupDocs.Annotation for .NET API позволяет разработчикам легко улучшать свои приложения с помощью возможности аннотировать текст или изображения в документе. Аннотации могут применяться в форме комментариев, пояснений с выделенными примечаниями, водяными знаками, полилиниями и многим другим. Annotator API поддерживает документы PDF, Microsoft Word DOCX, Excel XLSX и PowerPoint PPTX.

**Welcome to the GroupDocs.Conversion for .NET**

GroupDocs.Conversion для .NET — это расширенный API конвертации документов, разработанный для конвертации форматов документов из любого приложения .NET. Этот многофункциональный API позволяет вам работать с операциями конвертации документов вперед и назад для более чем 50 форматов документов и файлов изображений. Использование API конвертера документов очень просто, поскольку он использует только один метод с набором опций для конвертации между различными форматами.

**Welcome to the GroupDocs.Editor for .NET**

GroupDocs.Editor для .NET — это мощный API для редактирования документов с использованием HTML. API можно использовать с любым внешним, открытым или платным HTML-редактором. API редактора обрабатывает загрузку документов, преобразует их в HTML, предоставляет HTML внешнему пользовательскому интерфейсу и затем сохраняет HTML в исходный документ после обработки. Его также можно использовать для создания различных документов Microsoft Word, XML, форматов OpenDocument и файлов TXT.

**Welcome to the GroupDocs.Redaction for .NET**

GroupDocs.Redaction для .NET — это API для удаления конфиденциальной и секретной информации из документов различных форматов файлов. API предоставляет единый независимый от формата интерфейс для удаления конфиденциальной и секретной информации из документов. Используя API, вы можете удалить конфиденциальную информацию и содержимое из документа, используя различные методы редактирования, и сохранить документ в исходном формате, а также создать очищенный PDF-документ с растровыми изображениями исходных страниц.

**Welcome to the GroupDocs.Search for .NET**

GroupDocs.Search для .NET — это увлекательный API поиска документов. Он извлекает текст и метаданные из документов и выполняет расширенные операции поиска и индексации на основе нечетких и синонимических алгоритмов. API также поддерживает полнотекстовый поиск. **Welcome to the GroupDocs.Watermark for .NET**

GroupDocs.Watermark — это API для добавления водяных знаков в документы разных форматов. Он предоставляет эффективные методы добавления водяных знаков, которые позволяют добавлять водяные знаки, которые трудно удалить автоматически сторонними инструментами. Он также позволяет искать и удалять ранее добавленные водяные знаки популярных типов (включая водяные знаки, добавленные сторонними инструментами) в документе. API предоставляет простой и удобный набор методов для добавления, поиска и удаления водяных знаков в поддерживаемых форматах файлов.
