Его подсистема рендеринга помогает конечным пользователям отображать любой желаемый глиф или текст. Специальные глифы могут быть отображены путем реализации интерфейса с использованием простых графических функций (перемещение точки, рисование линии, кривой).

**Welcome to Aspose.TeX for Java**

Aspose.TeX — это библиотека Java для набора файлов TeX. Она может представлять входные данные TeX в различных графических форматах, таких как XPS, PDF, PNG, JPEG, TIFF и BMP. Она также выводит трассировочную информацию, как и любой другой движок TeX. API предоставляет различные способы передачи входных данных TeX и получения выходных данных. И, как расширенная функция, она позволяет вам создавать свой собственный формат TeX, если вы хотите набрать несколько документов, оформленных единообразно.
