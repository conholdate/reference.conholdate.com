GroupDocs.Total for Java — это компиляция всех API Java, предлагаемых GroupDocs. Он компилируется ежедневно, чтобы гарантировать, что он содержит самые последние версии каждого API GroupDocs.Total for Java.

**Welcome to the GroupDocs.Annotation for Java**

GroupDocs.Annotation for Java API позволяет разработчикам легко улучшать свои приложения с помощью возможности аннотировать текст или изображения в документе. Аннотации могут применяться в форме комментариев, пояснений с выделенными примечаниями, водяными знаками, полилиниями и многим другим. Annotator API поддерживает документы PDF, Microsoft Word DOCX, Excel XLSX и PowerPoint PPTX.

**Welcome to the GroupDocs.Assembly for Java**

GroupDocs.Assembly для Java — это API автоматизации документов и генерации отчетов, предназначенный для создания пользовательских документов из шаблонов. Этот API отчетов Java интеллектуально собирает заданные данные с определенным шаблоном документа и генерирует выходной документ на основе источника данных в том же формате, что и формат шаблона документа.

**Welcome to the GroupDocs.Conversion for Java**

GroupDocs.Conversion для Java — это передовой API конвертации документов, разработанный для конвертации форматов документов из любого приложения Java. Этот многофункциональный API позволяет вам работать с операциями конвертации документов вперед и назад для более чем 50 форматов документов и файлов изображений. Использовать API конвертера документов очень просто, поскольку он использует только один метод с набором опций для конвертации между различными форматами.

**Welcome to the GroupDocs.Editor for Java**

GroupDocs.Editor для Java — это мощный API для редактирования документов с использованием HTML. API можно использовать с любым внешним, открытым или платным HTML-редактором. API редактора обрабатывает загрузку документов, преобразует их в HTML, предоставляет HTML внешнему пользовательскому интерфейсу и затем сохраняет HTML в исходный документ после обработки. Его также можно использовать для создания различных документов Microsoft Word, XML, форматов OpenDocument и файлов TXT.

**Welcome to the GroupDocs.Redaction for Java**

GroupDocs.Redaction для Java — это API для удаления конфиденциальной и секретной информации из документов разных форматов файлов. API предоставляет единый независимый от формата интерфейс для удаления конфиденциальной и секретной информации из документов. С помощью API вы можете удалить конфиденциальную информацию и содержимое из документа, используя разные методы редактирования, и сохранить документ в исходном формате, а также создать очищенный PDF-документ с растровыми изображениями исходных страниц.

**Welcome to the GroupDocs.Search for Java**

GroupDocs.Search для Java — это увлекательный API поиска документов. Он извлекает текст и метаданные из документов и выполняет расширенные операции поиска и индексации на основе нечетких и синонимических алгоритмов. API также поддерживает полнотекстовый поиск. Поиск информации на основе полнотекстового поиска основан на индексах. Все документы, имеющие хотя бы одно совпадение во фразах, абзацах или словах, будут перечислены в качестве результата.

**Welcome to the GroupDocs.Watermark for Java**

GroupDocs.Watermark — это API для добавления водяных знаков в документы разных форматов. Он предоставляет эффективные методы добавления водяных знаков, которые позволяют добавлять водяные знаки, которые трудно удалить автоматически сторонними инструментами. Он также позволяет искать и удалять ранее добавленные водяные знаки популярных типов (включая водяные знаки, добавленные сторонними инструментами) в документе. API предоставляет простой и удобный набор методов для добавления, поиска и удаления водяных знаков в поддерживаемых форматах файлов.
