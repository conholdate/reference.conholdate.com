## การแนะนำ

ในบทช่วยสอนนี้ เราจะมาเรียนรู้วิธีการรวมไฟล์ DOC โดยใช้ GroupDocs.Merger สำหรับ .NET ซึ่งเป็น API ที่มีประสิทธิภาพที่ออกแบบมาสำหรับนักพัฒนาเพื่อรวม แยก และจัดการรูปแบบเอกสารต่างๆ ผ่านทางโปรแกรม รวมถึงไฟล์ DOC เราจะให้คำแนะนำทีละขั้นตอนแก่คุณเพื่ออำนวยความสะดวกให้กับกระบวนการนี้

## ข้อกำหนดเบื้องต้น

ก่อนที่จะเริ่มต้น ให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

1. Visual Studio: ติดตั้ง Visual Studio บนเครื่องพัฒนาของคุณ
2. GroupDocs.Merger สำหรับ .NET: ดาวน์โหลดไลบรารีจาก[เว็บไซต์](https://releases.groupdocs.com/merger/net/).
3. ความรู้พื้นฐานเกี่ยวกับ C#: ขอแนะนำให้มีความคุ้นเคยกับภาษาการเขียนโปรแกรม C#

## นำเข้าเนมสเปซที่จำเป็น

ในการทำงานกับ GroupDocs.Merger ขั้นแรกให้นำเข้าเนมสเปซที่จำเป็นลงในโปรเจ็กต์ C# ของคุณ:

```csharp
using System;
using System.IO;
```

## ขั้นตอนที่ 1: ระบุไดเรกทอรีผลลัพธ์

กำหนดไดเร็กทอรีเอาท์พุตที่จะบันทึกไฟล์ DOC ที่ผสานเข้าด้วยกัน:

```csharp
string outputFolder = "Your_Output_Directory"; // แทนที่ด้วยเส้นทางของคุณ
string outputFile = Path.Combine(outputFolder, "merged.doc");
```

 อย่าลืมเปลี่ยน`"Your_Output_Directory"` ด้วยเส้นทางจริงที่คุณต้องการบันทึกเอกสารที่ผสาน

## ขั้นตอนที่ 2: โหลดและรวมไฟล์ DOC ต้นทาง

ใช้โค้ดสั้นๆ ต่อไปนี้เพื่อโหลดไฟล์ DOC ต้นฉบับที่คุณต้องการรวม:

```csharp
using (var merger = new Merger("path_to_first_doc.doc"))
{
    // เพิ่มไฟล์ DOC อีกไฟล์เพื่อรวม
    merger.Join("path_to_second_doc.doc");

    // รวมไฟล์ DOC และบันทึกผลลัพธ์
    merger.Save(outputFile);
}
```


-  แทนที่`"path_to_first_doc.doc"` และ`"path_to_second_doc.doc"` พร้อมด้วยเส้นทางไฟล์แบบเต็มของไฟล์ DOC ที่คุณต้องการรวม
-  การ`merger.Join(...)` วิธีนี้ช่วยให้คุณสามารถเพิ่มไฟล์ DOC เพิ่มเติมลงในกระบวนการผสานได้
- `merger.Save(outputFile)` บันทึกเอกสารที่ผสานเป็น`merged.doc` ในโฟลเดอร์เอาท์พุตที่ระบุ
