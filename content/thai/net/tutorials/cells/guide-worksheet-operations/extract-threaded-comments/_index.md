## การแนะนำ

ในยุคดิจิทัล การจัดการและทำงานร่วมกันในเอกสารมีความสำคัญต่อเวิร์กโฟลว์ของเรา สเปรดชีต Excel ซึ่งมักเต็มไปด้วยข้อมูลและข้อมูลเชิงลึก มักมีคำอธิบายประกอบที่ให้บริบทหรือข้อเสนอแนะเพิ่มเติม ด้วย Aspose.Cells สำหรับ .NET การแยกและการจัดการความคิดเห็นแบบเธรดสามารถดำเนินไปได้อย่างราบรื่น บทช่วยสอนนี้จะแนะนำคุณตลอดขั้นตอนในการดึงความคิดเห็นแบบเธรดจากเวิร์กชีต Excel อย่างมีประสิทธิภาพ ไม่ว่าคุณจะมีประสบการณ์ด้านการเขียนโปรแกรมหรือไม่ก็ตาม 

## ข้อกำหนดเบื้องต้น
ก่อนที่จะเริ่มเขียนโค้ด ให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

1. ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับ C# และ .NET Framework ถือเป็นสิ่งสำคัญ เนื่องจากตัวอย่างโค้ดของเราจะอยู่ในภาษา C#
2. Visual Studio: ติดตั้ง Visual Studio บนเครื่องของคุณเพื่อรันโค้ด C#
3.  Aspose.Cells สำหรับ .NET: ดาวน์โหลดและติดตั้งไลบรารี Aspose.Cells จาก[เว็บไซต์อาโพส](https://releases.aspose.com/cells/net/).
4.  ตัวอย่างไฟล์ Excel: มีไฟล์ Excel ตัวอย่าง (เช่น`ThreadedCommentsSample.xlsx`) บันทึกไว้ในไดเร็กทอรีของคุณซึ่งมีข้อคิดเห็นแบบเธรดเพื่อการทดสอบ

## การนำเข้าแพ็คเกจที่จำเป็น
หากต้องการใช้ฟีเจอร์อันทรงพลังของ Aspose.Cells คุณต้องรวมเนมสเปซที่จำเป็นไว้ในโปรเจ็กต์ C# ของคุณ เพิ่มคำประกาศต่อไปนี้ที่จุดเริ่มต้นของไฟล์ C#:

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีแหล่งที่มา
ขั้นแรก ให้ระบุไดเรกทอรีที่ไฟล์ Excel ของคุณตั้งอยู่ ตรวจสอบให้แน่ใจว่าเส้นทางนั้นสอดคล้องกับตำแหน่งของไฟล์ในระบบของคุณ

```csharp
// ไดเรกทอรีแหล่งที่มา
string sourceDir = "Your Document Directory";
```
 แทนที่`"Your Document Directory\"` พร้อมเส้นทางจริงไปยังไฟล์ Excel ของคุณ

## ขั้นตอนที่ 2: สร้างวัตถุเวิร์กบุ๊ก
 ขั้นต่อไปสร้าง`Workbook` วัตถุในการโหลดและจัดการไฟล์ Excel ของคุณ

```csharp
//โหลดสมุดงาน
Workbook workbook = new Workbook(sourceDir + "ThreadedCommentsSample.xlsx");
```

## ขั้นตอนที่ 3: เข้าถึงแผ่นงาน
หลังจากโหลดเวิร์กบุ๊กแล้ว ให้เข้าถึงเวิร์กชีตเฉพาะที่ประกอบด้วยข้อคิดเห็นแบบเธรด สำหรับตัวอย่างนี้ เราจะเข้าถึงเวิร์กชีตแรก

```csharp
// เข้าถึงแผ่นงานแรก
Worksheet worksheet = workbook.Worksheets[0];
```

## ขั้นตอนที่ 4: รับความคิดเห็นแบบเธรด
ตอนนี้ ให้ดึงความคิดเห็นแบบเธรดจากเซลล์ที่ระบุ เราจะกำหนดเป้าหมายที่เซลล์ "A1" สำหรับตัวอย่างนี้

```csharp
// รับความคิดเห็นแบบเธรด
ThreadedCommentCollection threadedComments = worksheet.Comments.GetThreadedComments("A1");
```

## ขั้นตอนที่ 5: ทำซ้ำผ่านความคิดเห็น
โดยมีการรวบรวมความคิดเห็นแบบเธรดไว้ในมือ ให้วนซ้ำความคิดเห็นแต่ละข้อเพื่อดึงข้อมูลที่เกี่ยวข้อง เช่น ข้อความความคิดเห็นและชื่อผู้เขียน

```csharp
// วนซ้ำผ่านความคิดเห็นแบบเธรดแต่ละข้อ
foreach (ThreadedComment comment in threadedComments)
{
    Console.WriteLine("Comment: " + comment.Notes);
    Console.WriteLine("Author: " + comment.Author.Name);
}
```

## ขั้นตอนที่ 6: ยืนยันการดำเนินการสำเร็จ
สุดท้ายเรามายืนยันว่าโปรแกรมของเราทำงานสำเร็จ

```csharp
Console.WriteLine("ReadThreadedComments executed successfully.");
```
