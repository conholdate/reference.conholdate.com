**Welcome to the Aspose.Imaging for Java**

Aspose.Imaging para Java es una biblioteca de imágenes fácil de usar, de alto rendimiento y bien documentada. Proporciona el grupo de rutinas más flexible para crear, manipular y guardar imágenes en aplicaciones Java. Aspose.Imaging para Java admite la mayor cantidad de formatos de archivo de imágenes gráficas basadas en ráster y proporciona una amplia gama de algoritmos de compresión junto con numerosas operaciones de procesamiento de imágenes.

Aspose.Imaging para Java ofrece un amplio espectro de operaciones de procesamiento de imágenes tradicionales, como la conversión de una imagen de un formato a otro, el dibujo de objetos nativos, así como rutas gráficas y transformaciones geométricas como el escalado, la rotación y el volteo. Una característica exigente que ofrece Aspose.Imaging para Java es la manipulación de múltiples marcos de imágenes o la lectura y escritura de formatos de archivo que admitan múltiples marcos de imágenes.

**Welcome to Aspose.Font for Java**

Aspose.Font para Java es una biblioteca de carga y dibujo de fuentes. Admite varios formatos de fuentes, como TrueType (con colecciones TrueType), CFF, OpenType y Type1. La API proporciona una gran funcionalidad para cargar/guardar fuentes y proporcionar información sobre sus estructuras de datos junto con cualquier glifo compatible con todos los tipos de fuentes. Su subsistema de representación ayuda a los usuarios finales a representar cualquier glifo o texto deseado. Se pueden representar glifos especiales implementando una interfaz que utiliza una funcionalidad de gráficos simple (mover punto, dibujar línea, curva).
