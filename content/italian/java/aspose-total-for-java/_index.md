**Welcome to the Aspose.Imaging for Java**

Aspose.Imaging per Java è una libreria di imaging facile da usare, ad alte prestazioni e ben documentata. Fornisce il gruppo più flessibile di routine per creare, manipolare e salvare immagini nelle applicazioni Java. Aspose.Imaging per Java supporta i formati di file massimi da Raster based graphical Imaging e fornisce un'ampia gamma di algoritmi di compressione insieme a numerose operazioni di elaborazione delle immagini.

Aspose.Imaging per Java offre un ampio spettro di operazioni di elaborazione delle immagini tradizionali, come la conversione di un'immagine da un formato a un altro, il disegno di oggetti nativi, nonché GraphicsPath e trasformazioni geometriche come ridimensionamento, rotazione e capovolgimento. Una funzionalità esigente fornita da Aspose.Imaging per Java è la manipolazione di più frame di immagini o la lettura e la scrittura di formati di file che supportano più frame di immagini.
