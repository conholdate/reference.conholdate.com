## مقدمة

مرحبًا بك في عالم معالجة ملفات PDF المثير باستخدام Aspose.PDF لـ .NET! سواء كنت تقوم بتحسين المستندات للاستخدام المهني أو المشاريع الشخصية أو أي شيء بينهما، فأنت في المكان المناسب. في هذا الدليل، سنستكشف ميزة عملية لـ Aspose.PDF: إضافة تعليقات بالحبر إلى ملفات PDF الخاصة بك. هذه الوظيفة مثالية لدمج الملاحظات أو التوقيعات المكتوبة بخط اليد، مما يجعل مستنداتك أكثر تفاعلية وجاذبية.

## المتطلبات الأساسية

قبل أن ننتقل إلى الكود، دعنا نتأكد من إعداد كل شيء:

1. .NET Framework: تأكد من تثبيت .NET Framework على جهازك. يعمل Aspose.PDF بسلاسة مع إصدارات مختلفة، بما في ذلك .NET Core.
2.  مكتبة Aspose.PDF: قم بتنزيل مكتبة Aspose.PDF لـ .NET والرجوع إليها في مشروعك. يمكنك الحصول على أحدث إصدار من[رابط التحميل](https://releases.aspose.com/pdf/net/).
3. محرر الكود: على الرغم من أنه يمكنك استخدام أي محرر كود، إلا أنه يوصى بشدة باستخدام Visual Studio بسبب واجهته سهلة الاستخدام مع تطبيقات .NET.
4. المعرفة الأساسية بلغة C#: ستساعدك المعرفة بلغة C# على التنقل بين أمثلة الترميز بسلاسة.
5. إعداد بيئة التطوير: تأكد من تكوين IDE الخاص بك لمشاريع .NET ومن أنك قمت بالرجوع إلى مكتبة Aspose.PDF بشكل صحيح.

بمجرد توفر هذه المتطلبات الأساسية لديك، ستكون جاهزًا لبدء إضافة التعليقات التوضيحية بالحبر إلى ملفات PDF الخاصة بك!

## استيراد الحزم الضرورية

قبل الخوض في الترميز، دعنا نستورد الحزم المطلوبة. في أعلى ملف C#، أضف عبارات الاستخدام التالية:

```csharp
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
using System;
using System.Collections.Generic;
```

ستوفر هذه البيانات إمكانية الوصول إلى جميع الفئات والطرق اللازمة للعمل مع تعليقات PDF.

دعونا نقوم بتقسيم عملية إضافة تعليق حبر إلى مستند PDF الخاص بك إلى خطوات واضحة.

## الخطوة 1: إعداد المستند والدليل

أولاً، قم بإنشاء المستند والمسار لحفظ ملف الإخراج:

```csharp
string dataDir = "YOUR DATA DIRECTORY";
Document doc = new Document();
```

 هنا،`dataDir` يشير إلى الدليل الذي سيتم حفظ ملف PDF الناتج فيه، ونقوم بإنشاء مثيل جديد`Document` كائن للتحرير.

## الخطوة 2: إضافة صفحة إلى مستندك

بعد ذلك، أضف صفحة إلى المستند الذي قمت بإنشائه حديثًا:

```csharp
Page pdfPage = doc.Pages.Add();
```

يتطلب كل ملف PDF صفحة واحدة على الأقل، لذا فإن هذه الخطوة ضرورية.

## الخطوة 3: تحديد مستطيل الرسم

الآن، قم بتحديد المكان على الصفحة الذي ستضع فيه تعليقات الحبر الخاصة بك:

```csharp
System.Drawing.Rectangle drect = new System.Drawing.Rectangle
{
    Height = (int)pdfPage.Rect.Height,
    Width = (int)pdfPage.Rect.Width,
    X = 0,
    Y = 0
};
Aspose.Pdf.Rectangle arect = Aspose.Pdf.Rectangle.FromRect(drect);
```

 هذا الكود ينشئ`Rectangle` كائن يحدد المنطقة على الصفحة لتعليق الحبر الخاص بك، والذي يناسب الصفحة بأكملها.

## الخطوة 4: تحضير نقاط الحبر

بعد ذلك، قم بتحديد النقاط التي ستشكل شرح الحبر الخاص بك:

```csharp
IList<Point[]> inkList = new List<Point[]>();
Aspose.Pdf.Point[] arrpt = new Aspose.Pdf.Point[3];
inkList.Add(arrpt);
arrpt[0] = new Aspose.Pdf.Point(100, 800);
arrpt[1] = new Aspose.Pdf.Point(200, 800);
arrpt[2] = new Aspose.Pdf.Point(200, 700);
```

تنشئ هذه الكتلة قائمة بمصفوفات النقاط، حيث تمثل كل مصفوفة مجموعة من النقاط لضربة الحبر الخاصة بك. هنا، نقوم بتحديد ثلاث نقاط تشكل مثلثًا، ولكن لا تتردد في تعديل الإحداثيات لتناسب تصميمك.

## الخطوة 5: إنشاء تعليق بالحبر

بعد تحديد النقاط الخاصة بك، قم بإنشاء تعليق بالحبر:

```csharp
InkAnnotation ia = new InkAnnotation(pdfPage, arect, inkList)
{
    Title = "Your Title",
    Color = Aspose.Pdf.Color.LightBlue,
    CapStyle = CapStyle.Rounded
};
```

 نحن ننشئ`InkAnnotation` الكائن، يمرر الصفحة والمستطيل ونقاط الحبر. قم بتخصيص خصائص مثل`Title`, `Color` ، و`CapStyle` لتناسب احتياجاتك!

## الخطوة 6: تعيين الحدود والتعتيم

لتجعل تعليقك مميزًا، دعنا نقوم بتنسيقه:

```csharp
Border border = new Border(ia)
{
    Width = 25
};
ia.Border = border;
ia.Opacity = 0.5;
```

يضيف هذا الكود حدودًا بعرض محدد ويضبط تعتيم التعليق التوضيحي لجعله شفافًا إلى حد ما.

## الخطوة 7: إضافة التعليقات التوضيحية إلى الصفحة

الآن، أضف تعليقك إلى صفحة PDF:

```csharp
pdfPage.Annotations.Add(ia);
```

يضيف هذا السطر تعليق الحبر إلى مجموعة تعليقات الصفحة.

## الخطوة 8: حفظ المستند

وأخيرًا، احفظ مستندك المعدّل:

```csharp
dataDir = dataDir + "AddInkAnnotation_out.pdf";
doc.Save(dataDir);
Console.WriteLine("\nInk annotation added successfully.\nFile saved at " + dataDir);
```

 هنا نقوم بالتعديل`dataDir` لتضمين اسم ملف الإخراج وحفظ المستند. ستظهر لك رسالة تأكيد تخبرك بأن كل شيء سار بسلاسة.
