## مقدمة

عند العمل مع جداول بيانات Excel كبيرة الحجم، يعد التحكم في تخطيط الطباعة أمرًا بالغ الأهمية لتحقيق الوضوح والتنظيم. يوفر Aspose.Cells for .NET ميزات قوية تتيح لك تخصيص إعدادات الطباعة لأوراق العمل الخاصة بك دون عناء. في هذا الدليل، سنستعرض الخطوات اللازمة لتعيين ترتيب الصفحات للطباعة عبر الصفوف أولاً ثم عبر الأعمدة.

3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة بلغة C# على فهم مقتطفات التعليمات البرمجية.

 يمكنك أيضًا تجربة[Aspose.Cells لـ .NET مع نسخة تجريبية مجانية](https://releases.aspose.com/) أو الحصول على[رخصة مؤقتة](https://purchase.aspose.com/temporary-license/) للحصول على إمكانية الوصول إلى الميزات الكاملة.

## استيراد الحزم الضرورية

ابدأ باستيراد المساحات المطلوبة للوصول إلى وظائف Aspose.Cells:

```csharp
using System.IO;
using Aspose.Cells;
using System;
```

## الخطوة 1: إنشاء مصنف

أولاً، قم بإنشاء مثيل مصنف جديد، والذي يمثل ملف Excel الخاص بك.

```csharp
// إنشاء كائن مصنف جديد
Workbook workbook = new Workbook();
```

يقوم هذا السطر بتهيئة مصنف Excel فارغًا وجاهزًا للتخصيص.

## الخطوة 2: الوصول إلى صفحة إعداد ورقة العمل

 لضبط إعدادات الطباعة، قم بالوصول إلى`PageSetup` هدف ورقة العمل.

```csharp
// الوصول إلى صفحة إعداد ورقة العمل الأولى
PageSetup pageSetup = workbook.Worksheets[0].PageSetup;
```

 هنا، نسترد`PageSetup` للورقة العمل الأولى، حيث سنقوم بتكوين تخطيط الطباعة.

## الخطوة 3: اضبط ترتيب الصفحات على OverThenDown

الآن، لنقم بتعيين ترتيب الصفحات. بشكل افتراضي، يقوم برنامج Excel بطباعة كل عمود أولاً؛ وسنقوم بتغيير ذلك لطباعة كل الصفوف أولاً.

```csharp
// تعيين ترتيب الطباعة إلى OverThenDown
pageSetup.Order = PrintOrderType.OverThenDown;
```

يضمن هذا الإعداد أنه عند الطباعة، تتدفق البيانات أفقياً قبل الانتقال إلى الصف التالي، وهو أمر مفيد بشكل خاص لمجموعات البيانات الواسعة.

## الخطوة 4: احفظ المصنف

وأخيرًا، احفظ المصنف الخاص بك لتطبيق التغييرات.

```csharp
// تحديد المسار لحفظ المصنف
string dataDir = "Your Document Directory/";
// حفظ المصنف
workbook.Save(dataDir + "SetPageOrder_out.xls");
```

 يستبدل`"Your Document Directory"`مع مسار الملف المطلوب. يمكنك أيضًا حفظه بتنسيقات أخرى، مثل`.xlsx`، عن طريق تغيير امتداد الملف.

 بجانب`OverThenDown` ، يمكنك استخدام`DownThenOver`، والتي تطبع الأعمدة أولاً قبل الانتقال عبر الصفوف.
