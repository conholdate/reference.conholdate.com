Aspose.Imaging for Java é uma biblioteca de imagens fácil de usar, de alto desempenho e bem documentada. Um recurso exigente fornecido pelo Aspose.Imaging for Java é manipular vários quadros de imagem ou ler e escrever formatos de arquivo que suportam vários quadros de imagem.

Aspose.Font para Java é uma biblioteca de carregamento e desenho de fontes. Ela suporta múltiplos formatos de fonte como TrueType (com coleções TrueType), CFF, OpenType e Type1. Seu subsistema de renderização ajuda os usuários finais a renderizar qualquer glifo ou texto desejado.
