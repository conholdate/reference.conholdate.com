## परिचय

हमारी बढ़ती बहुभाषी दुनिया में, विभिन्न भाषा वरीयताओं के अनुरूप दस्तावेजों को अनुकूलित करना आवश्यक है। यदि आप .NET के लिए Aspose.Words के साथ काम कर रहे हैं, तो यह ट्यूटोरियल आपको अपने Word दस्तावेज़ों में रूसी को डिफ़ॉल्ट संपादन भाषा के रूप में सेट करने की प्रक्रिया के माध्यम से मार्गदर्शन करेगा। 

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1.  .NET के लिए Aspose.Words: लाइब्रेरी को यहाँ से डाउनलोड करें[एस्पोज रिलीज](https://releases.aspose.com/words/net/) पृष्ठ.
2. विकास वातावरण: .NET अनुप्रयोगों को कोडिंग और चलाने के लिए विजुअल स्टूडियो जैसे IDE की सिफारिश की जाती है।
3. C# का बुनियादी ज्ञान: इस ट्यूटोरियल को प्रभावी ढंग से समझने के लिए C# और .NET फ्रेमवर्क से परिचित होना आवश्यक है।

## आवश्यक नामस्थान आयात करना

Word दस्तावेज़ों में हेरफेर करने के लिए, आपको अपने प्रोजेक्ट में निम्नलिखित नामस्थानों को आयात करना होगा:

```csharp
using Aspose.Words;
using Aspose.Words.Loading;
```

## चरण 1: लोडऑप्शन कॉन्फ़िगर करें

 पहला कदम स्थापित करना है`LoadOptions`, जो आपको अपने दस्तावेज़ के लिए डिफ़ॉल्ट संपादन भाषा निर्दिष्ट करने की अनुमति देता है.

### LoadOptions इंस्टेंस बनाएं

 इसका एक उदाहरण बनाकर शुरू करें`LoadOptions`:

```csharp
LoadOptions loadOptions = new LoadOptions();
```

### डिफ़ॉल्ट संपादन भाषा को रूसी पर सेट करें

इसके बाद, सेट करें`DefaultEditingLanguage` संपत्ति रूसी के लिए:

```csharp
loadOptions.LanguagePreferences.DefaultEditingLanguage = EditingLanguage.Russian;
```

यह कॉन्फ़िगरेशन Aspose.Words को बताता है कि जब भी दस्तावेज़ इन विकल्पों के साथ लोड किया जाता है, तो रूसी को डिफ़ॉल्ट संपादन भाषा के रूप में माना जाता है।

## चरण 2: अपना दस्तावेज़ लोड करें

 अब, आपको कॉन्फ़िगर किए गए का उपयोग करके Word दस्तावेज़ को लोड करना होगा`LoadOptions`.

### दस्तावेज़ पथ निर्दिष्ट करें

अपने दस्तावेज़ का पथ निर्धारित करें:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

### LoadOptions के साथ दस्तावेज़ लोड करें

 फिर, दस्तावेज़ को लोड करें`Document` निर्माता:

```csharp
Document doc = new Document(dataDir + "No default editing language.docx", loadOptions);
```

यह चरण सुनिश्चित करता है कि लोड किए गए दस्तावेज़ के लिए रूसी भाषा को डिफ़ॉल्ट संपादन भाषा के रूप में सेट किया गया है।

## चरण 3: डिफ़ॉल्ट संपादन भाषा सत्यापित करें

दस्तावेज़ लोड करने के बाद, यह पुष्टि करना महत्वपूर्ण है कि डिफ़ॉल्ट संपादन भाषा सही ढंग से रूसी पर सेट है।

### डिफ़ॉल्ट फ़ॉन्ट का LocaleId प्राप्त करें

 लाओ`LocaleId` दस्तावेज़ की डिफ़ॉल्ट फ़ॉन्ट शैली का:

```csharp
int localeId = doc.Styles.DefaultFont.LocaleId;
```

### LocaleId जांचें

 अंत में, तुलना करें`LocaleId` यह देखने के लिए कि क्या यह रूसी भाषा से मेल खाता है:

```csharp
Console.WriteLine(
    localeId == (int)EditingLanguage.Russian
        ? "The document's default editing language is set to Russian."
        : "The document's default language is not set to Russian.");
```

यह आउटपुट आपको सूचित करेगा कि डिफ़ॉल्ट संपादन भाषा सफलतापूर्वक रूसी पर सेट की गई है या नहीं।

## निष्कर्ष

Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ में रूसी को डिफ़ॉल्ट संपादन भाषा के रूप में सेट करना एक सीधी प्रक्रिया है।`LoadOptions`दस्तावेज़ लोड करने और भाषा सेटिंग्स को सत्यापित करने के बाद, आप अपने दस्तावेज़ों को अपने दर्शकों की भाषाई आवश्यकताओं को प्रभावी ढंग से पूरा करने के लिए तैयार कर सकते हैं।

## अक्सर पूछे जाने वाले प्रश्न

### .NET के लिए Aspose.Words क्या है?

.NET के लिए Aspose.Words .NET अनुप्रयोगों के भीतर Word दस्तावेज़ों को प्रोग्रामेटिक रूप से बनाने, हेरफेर करने और परिवर्तित करने के लिए एक व्यापक लाइब्रेरी है।

### मैं .NET के लिए Aspose.Words कैसे डाउनलोड करूं?

 आप .NET के लिए Aspose.Words को यहां से डाउनलोड कर सकते हैं[एस्पोज रिलीज](https://releases.aspose.com/words/net/) पृष्ठ.

###  क्या है`LoadOptions` used for?

`LoadOptions` आपको दस्तावेज़ लोड करने के लिए विभिन्न विकल्प निर्दिष्ट करने की अनुमति देता है, जिसमें डिफ़ॉल्ट संपादन भाषा सेट करना भी शामिल है।

### क्या मैं अन्य भाषाओं को डिफ़ॉल्ट संपादन भाषा के रूप में सेट कर सकता हूँ?

 हां, आप उपयुक्त निर्दिष्ट करके Aspose.Words द्वारा समर्थित किसी भी भाषा को सेट कर सकते हैं`EditingLanguage` मूल्य`DefaultEditingLanguage`.

### मैं .NET के लिए Aspose.Words का समर्थन कैसे प्राप्त कर सकता हूं?

 सहायता के लिए, यहां जाएं[Aspose समर्थन](https://forum.aspose.com/c/words/8)मंच, जहां आप प्रश्न पूछ सकते हैं और समुदाय और Aspose डेवलपर्स से सहायता प्राप्त कर सकते हैं।
