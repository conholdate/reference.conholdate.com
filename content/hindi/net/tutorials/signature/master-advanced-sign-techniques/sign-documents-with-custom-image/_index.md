## परिचय

इस ट्यूटोरियल में, आप सीखेंगे कि छवियों के साथ दस्तावेज़ों पर हस्ताक्षर करने के लिए .NET के लिए GroupDocs.Signature का उपयोग कैसे करें। दस्तावेज़ हस्ताक्षर आपकी फ़ाइलों की प्रामाणिकता और सुरक्षा को बढ़ाता है, यह सुनिश्चित करता है कि वे छेड़छाड़-रहित और कानूनी रूप से बाध्यकारी हैं। अपने .NET अनुप्रयोगों में दस्तावेज़ हस्ताक्षर कार्यक्षमता को एकीकृत करके, आप अपने वर्कफ़्लो को काफी हद तक सुव्यवस्थित कर सकते हैं।

## आवश्यक शर्तें

ट्यूटोरियल में आगे बढ़ने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1.  .NET के लिए GroupDocs.Signature: .NET के लिए GroupDocs.Signature को डाउनलोड करें और इंस्टॉल करें[वेबसाइट](https://releases.groupdocs.com/signature/net/).
2. .NET विकास वातावरण: .NET विकास के लिए कार्य वातावरण स्थापित करें।

## नामस्थान आयात करें

आवश्यक क्लासों और विधियों तक पहुंचने के लिए, अपने प्रोजेक्ट में आवश्यक नामस्थानों को आयात करके शुरू करें:

```csharp
using System;
using System.IO;
using GroupDocs.Signature;
using GroupDocs.Signature.Domain;
using GroupDocs.Signature.Options;
```

## चरण 1: दस्तावेज़ लोड करें

उस दस्तावेज़ का पथ निर्दिष्ट करें जिस पर आप हस्ताक्षर करना चाहते हैं। उदाहरण के लिए, PDF फ़ाइल लोड करने के लिए:

```csharp
string filePath = "sample.pdf";
```

## चरण 2: हस्ताक्षर छवि निर्दिष्ट करें

उस हस्ताक्षर छवि का पथ परिभाषित करें जिसका आप उपयोग करना चाहते हैं:

```csharp
string imagePath = "signature_handwrite.jpg";
```

## चरण 3: आउटपुट फ़ाइल पथ सेट करें

निर्धारित करें कि आप हस्ताक्षरित दस्तावेज़ को कहाँ सहेजना चाहते हैं:

```csharp
string outputFilePath = Path.Combine("Your Document Directory", "SignWithImage", "SignedDocument.pdf");
```

## चरण 4: हस्ताक्षर ऑब्जेक्ट को आरंभ करें

 इसका एक उदाहरण बनाएं`Signature`क्लास, दस्तावेज़ फ़ाइल पथ में पासिंग:

```csharp
using (Signature signature = new Signature(filePath))
{
    // अतिरिक्त कोड यहाँ जाएगा
}
```

## चरण 5: छवि हस्ताक्षर विकल्प कॉन्फ़िगर करें

दस्तावेज़ पर हस्ताक्षर करने के लिए विकल्प सेट करें। यहाँ, आप हस्ताक्षर की स्थिति और यह निर्दिष्ट कर सकते हैं कि क्या यह सभी पृष्ठों पर दिखाई देना चाहिए:

```csharp
ImageSignOptions options = new ImageSignOptions(imagePath)
{
    Left = 50,   // क्षैतिज स्थिति
    Top = 50,    // ऊर्ध्वाधर स्थिति
    AllPages = true // सभी पृष्ठों पर हस्ताक्षर करें
};
```

## चरण 6: दस्तावेज़ पर हस्ताक्षर करें

दस्तावेज़ पर हस्ताक्षर करने के लिए कॉन्फ़िगर किए गए विकल्पों का उपयोग करें:

```csharp
SignResult result = signature.Sign(outputFilePath, options);
```

## चरण 7: परिणाम प्रदर्शित करें

अंत में, उपयोगकर्ता को हस्ताक्षर प्रक्रिया की सफलता के बारे में सूचित करें, जिसमें हस्ताक्षरित दस्तावेज़ का स्थान भी शामिल है:

```csharp
Console.WriteLine($"\nSource document signed successfully with {result.Succeeded.Count} signature(s).\nFile saved at {outputFilePath}.");
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए GroupDocs.Signature के साथ .NET अनुप्रयोगों में छवियों का उपयोग करके दस्तावेज़ों पर हस्ताक्षर करने का तरीका बताया। दस्तावेज़ पर हस्ताक्षर करना आपकी फ़ाइलों की प्रामाणिकता और सुरक्षा बनाए रखने के लिए आवश्यक है, जो आपकी दस्तावेज़ प्रबंधन क्षमताओं को महत्वपूर्ण रूप से बढ़ाता है।

## अक्सर पूछे जाने वाले प्रश्न

### क्या मैं एक ही दस्तावेज़ में एकाधिक हस्ताक्षर छवियों का उपयोग कर सकता हूँ?

हां, आप कई छवियों का उपयोग करके दस्तावेज़ पर हस्ताक्षर कर सकते हैं। बस आवश्यकतानुसार प्रत्येक छवि के लिए हस्ताक्षर प्रक्रिया को दोहराएं।

### क्या GroupDocs.Signature for .NET सभी दस्तावेज़ प्रकारों के साथ संगत है?

.NET के लिए GroupDocs.Signature पीडीएफ, वर्ड, एक्सेल और अधिक सहित विभिन्न दस्तावेज़ प्रारूपों का समर्थन करता है।

### क्या मैं हस्ताक्षर के स्वरूप को अनुकूलित कर सकता हूँ?

बिल्कुल! आप हस्ताक्षर के स्वरूप के विभिन्न पहलुओं को समायोजित कर सकते हैं, जैसे आकार, स्थिति, पारदर्शिता, और बहुत कुछ।

### क्या परीक्षण हेतु कोई परीक्षण संस्करण उपलब्ध है?

हां, आप खरीदारी करने से पहले इसकी कार्यक्षमता का पता लगाने के लिए वेबसाइट से एक निःशुल्क परीक्षण संस्करण डाउनलोड कर सकते हैं।

 तकनीकी सहायता के लिए कृपया यहां जाएं[GroupDocs.Signature फ़ोरम](https://forum.groupdocs.com/c/signature/13).
