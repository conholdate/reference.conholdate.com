## आवश्यक नामस्थान आयात करें

अपनी C# फ़ाइल के शीर्ष पर, Aspose.PDF नामस्थान आयात करें:

```csharp
using System;
using System.IO;
using Aspose.Pdf;
```

सब कुछ सेट हो जाने के बाद, आइए अपने ट्यूटोरियल के मुख्य भाग की ओर बढ़ें!

हम बुकमार्क्स निकालने की प्रक्रिया को प्रबंधनीय चरणों में विभाजित करेंगे।

## चरण 1: अपनी दस्तावेज़ निर्देशिका निर्दिष्ट करें

सबसे पहले, अपने दस्तावेज़ निर्देशिका का पथ सेट करें जहां आपकी पीडीएफ फाइल स्थित है:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // अपने वास्तविक पथ से प्रतिस्थापित करें
```

## चरण 2: पीडीएफ दस्तावेज़ खोलें

इसके बाद, हम Aspose.PDF लाइब्रेरी का उपयोग करके PDF दस्तावेज़ खोलेंगे:

```csharp
// पीडीएफ दस्तावेज़ खोलें
Document pdfDocument = new Document(dataDir + "GetBookmarks.pdf");
```

सुनिश्चित करें कि फ़ाइल का नाम आपकी निर्देशिका में मौजूद नाम से मेल खाता हो।

## चरण 3: बुकमार्क्स के माध्यम से लूप करें

अब चूंकि पीडीएफ दस्तावेज़ खुल गया है, तो आइए सभी बुकमार्क्स को देखें और प्रासंगिक जानकारी निकालें:

```csharp
// सभी बुकमार्क पर जाएँ
foreach (OutlineItemCollection outlineItem in pdfDocument.Outlines)
{
    Console.WriteLine($"Title: {outlineItem.Title}");
    Console.WriteLine($"Italic: {outlineItem.Italic}");
    Console.WriteLine($"Bold: {outlineItem.Bold}");
    Console.WriteLine($"Color: {outlineItem.Color}");
    Console.WriteLine(new string('-', 20)); // स्पष्टता के लिए विभाजक
}
```

इस स्निपेट में, हम प्रत्येक के माध्यम से पुनरावृत्ति करते हैं`OutlineItemCollection` में`pdfDocument.Outlines`, प्रत्येक बुकमार्क का शीर्षक, इटैलिक स्थिति, बोल्ड स्थिति और रंग प्रिंट करना। यह आपको आपके पीडीएफ में बुकमार्क का एक व्यापक दृश्य देता है।
