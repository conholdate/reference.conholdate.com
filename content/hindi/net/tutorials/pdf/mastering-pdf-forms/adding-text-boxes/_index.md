## परिचय

आज के डिजिटल परिदृश्य में, इंटरैक्टिव दस्तावेज़ों के माध्यम से उपयोगकर्ता अनुभव को बेहतर बनाना आवश्यक है। इंटरैक्टिव PDF फ़ॉर्म न केवल डेटा संग्रह को सुव्यवस्थित करते हैं, बल्कि उपयोगकर्ताओं को एक ऐसे तरीके से जोड़ते हैं जो स्थिर दस्तावेज़ नहीं कर सकते। Aspose.PDF for .NET एक शक्तिशाली लाइब्रेरी है जिसे डेवलपर्स को PDF दस्तावेज़ों में विभिन्न फ़ॉर्म फ़ील्ड को आसानी से एकीकृत करने में मदद करने के लिए डिज़ाइन किया गया है। इनमें से, टेक्स्ट बॉक्स संरचित तरीके से उपयोगकर्ता इनपुट एकत्र करने के लिए विशेष रूप से उपयोगी हैं। इस ट्यूटोरियल में, हम Aspose.PDF for .NET का उपयोग करके PDF में टेक्स्ट बॉक्स जोड़ने की प्रक्रिया से गुजरेंगे, यह सुनिश्चित करते हुए कि आपको प्रत्येक चरण की व्यापक समझ है।

2.  .NET के लिए Aspose.PDF स्थापित: से Aspose.PDF लाइब्रेरी डाउनलोड और स्थापित करें[साइट](https://releases.aspose.com/pdf/net/).
3. विकास वातावरण: कोडिंग और परीक्षण के लिए विजुअल स्टूडियो जैसे IDE का उपयोग करें।
4. .NET फ्रेमवर्क: सुनिश्चित करें कि आपके पास .NET फ्रेमवर्क का संगत संस्करण स्थापित है।

इन पूर्व-आवश्यकताओं के साथ, हम कोडिंग शुरू करने के लिए तैयार हैं!

### अपना IDE खोलें

अपना पसंदीदा विकास वातावरण लॉन्च करें (विजुअल स्टूडियो अनुशंसित है)।

"एक नया प्रोजेक्ट बनाएं" का चयन करके और सरलता के लिए कंसोल एप्लिकेशन टेम्पलेट चुनकर एक नया C# प्रोजेक्ट सेट अप करें।

### Aspose.PDF पैकेज स्थापित करें

NuGet पैकेज मैनेजर का उपयोग करके Aspose.PDF लाइब्रेरी को अपने प्रोजेक्ट में एकीकृत करें। पैकेज मैनेजर कंसोल में, चलाएँ:

```bash
Install-Package Aspose.PDF
```

## Aspose.PDF नामस्थान आयात करें

 आपकी मुख्य प्रोग्राम फ़ाइल के शीर्ष पर (आमतौर पर`Program.cs`), निम्नलिखित नामस्थान शामिल करें:

```csharp
using System.IO;
using System;
using Aspose.Pdf.Forms;
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
```

यह सेटअप आपको आगे आने वाले रोमांचक कार्यों के लिए तैयार करता है!

अब जब हमारे पास सब कुछ तैयार है, तो आइए अपने पीडीएफ दस्तावेज़ में टेक्स्ट बॉक्स जोड़ने के चरणों को तोड़ दें।

## चरण 1: अपनी दस्तावेज़ निर्देशिका निर्धारित करें

 सबसे पहले, वह निर्देशिका निर्दिष्ट करें जहाँ आपका PDF दस्तावेज़ स्थित है।`"YOUR DOCUMENT DIRECTORY"` वास्तविक पथ के साथ:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: पीडीएफ दस्तावेज़ खोलें 

 पीडीएफ फाइल को एक उदाहरण में लोड करें`Document` कक्षा:

```csharp
Document pdfDocument = new Document(dataDir + "TextField.pdf");
```

 यह सुनिश्चित करें कि`"TextField.pdf"` आपके निर्दिष्ट निर्देशिका में मौजूद है.

## चरण 3: टेक्स्ट बॉक्स फ़ील्ड बनाएँ

अब, आइए टेक्स्ट बॉक्स फ़ील्ड बनाएं:

```csharp
TextBoxField textBoxField = new TextBoxField(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(100, 200, 300, 300));
```

-  एक नया`TextBoxField` ऑब्जेक्ट आपके पीडीएफ के दूसरे पृष्ठ के लिए आरंभीकृत है।
- `Rectangle` पैरामीटर निर्देशांक (x1, y1, x2, y2) का उपयोग करके टेक्स्ट बॉक्स की स्थिति और आकार निर्दिष्ट करता है।

## चरण 4: टेक्स्ट बॉक्स फ़ील्ड के लिए गुण सेट करें 

अपने टेक्स्ट बॉक्स को निम्नलिखित गुणों के साथ अनुकूलित करें:

```csharp
textBoxField.PartialName = "textbox1";
textBoxField.Value = "Text Box";
```

- `PartialName` टेक्स्ट बॉक्स को एक विशिष्ट पहचान देता है.
- `Value` बॉक्स के अंदर दिखाई देने वाला डिफ़ॉल्ट टेक्स्ट सेट करता है.

## चरण 5: बॉर्डर को अनुकूलित करें

आइए अपने टेक्स्ट बॉक्स की बॉर्डर को अनुकूलित करके उसके स्वरूप को बेहतर बनाएं:

```csharp
Border border = new Border(textBoxField);
border.Width = 5; 
border.Dash = new Dash(1, 1);
textBoxField.Border = border;
textBoxField.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
```

- एक बॉर्डर बनाता है और उसकी चौड़ाई निर्धारित करता है.
- बॉर्डर पर धराशायी शैली लागू करता है.
- टेक्स्ट बॉक्स को हरा रंग प्रदान करता है।

## चरण 6: दस्तावेज़ में टेक्स्ट बॉक्स जोड़ें

अब, हम अपने PDF दस्तावेज़ में टेक्स्ट बॉक्स फ़ील्ड जोड़ेंगे:

```csharp
pdfDocument.Form.Add(textBoxField, 1);
```

यह पंक्ति पीडीएफ के प्रथम पृष्ठ में टेक्स्ट बॉक्स को सम्मिलित करती है।

## चरण 7: संशोधित पीडीएफ को सहेजें

अंत में, अपने परिवर्तनों को निम्नलिखित कोड के साथ सहेजें:

```csharp
dataDir = dataDir + "TextBox_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nTextbox field added successfully.\nFile saved at " + dataDir);
```

यह स्निपेट संशोधित PDF को नए नाम से सहेजता है। अपने नए बनाए गए PDF के लिए आउटपुट पथ की जाँच करें!

## निष्कर्ष

बधाई हो! आपने .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में सफलतापूर्वक एक टेक्स्ट बॉक्स जोड़ लिया है। यह प्रक्रिया न केवल आपके PDF की अन्तरक्रियाशीलता को बढ़ाती है बल्कि उपयोगकर्ता सहभागिता में भी उल्लेखनीय सुधार करती है। चाहे आप उपयोगकर्ता इनपुट एकत्र कर रहे हों, सर्वेक्षण कर रहे हों या फ़ॉर्म बना रहे हों, टेक्स्ट बॉक्स आपके PDF दस्तावेज़ों की कार्यक्षमता को बढ़ा सकते हैं। अगली बार जब आप PDF बनाएँ, तो इंटरैक्टिव फ़ील्ड की शक्ति को याद रखें और Aspose.PDF के साथ उन्हें लागू करना कितना आसान है।
