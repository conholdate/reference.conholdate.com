## Εισαγωγή

Όταν εργάζεστε με αρχεία Excel, η διαχείριση σχολίων μπορεί να είναι απαραίτητη για τη συνεργασία και την παρακολούθηση σχολίων. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε στη διαδικασία ανάγνωσης του δημιουργημένου χρόνου των σχολίων με νήματα σε ένα φύλλο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για .NET. Αυτό το ισχυρό εργαλείο παρέχει έναν αποτελεσματικό τρόπο αλληλεπίδρασης με αρχεία Excel, επιτρέποντας στους προγραμματιστές να εξάγουν λεπτομερείς πληροφορίες από σχόλια, κάτι που είναι ιδιαίτερα χρήσιμο για την παρακολούθηση του χρόνου των σχολίων σε συνεργατικά σενάρια.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, είναι σημαντικό να βεβαιωθείτε ότι το περιβάλλον ανάπτυξής σας έχει ρυθμιστεί σωστά για να χρησιμοποιήσετε το Aspose.Cells για .NET. Εδώ είναι τι θα χρειαστείτε:

1.  Aspose.Cells για .NET: Θα χρειαστείτε εγκατεστημένη τη βιβλιοθήκη Aspose.Cells. Μπορείτε να λάβετε την πιο πρόσφατη έκδοση από το[Aspose website](https://releases.aspose.com/cells/net/).
2. IDE: Visual Studio (ή οποιοδήποτε .NET IDE της επιλογής σας) για να γράψετε και να εκτελέσετε τον κώδικά σας.
3. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα διευκολύνει την παρακολούθηση των παραδειγμάτων.
4.  Αρχείο Excel: Για αυτό το σεμινάριο, θα χρησιμοποιήσουμε ένα αρχείο Excel με το όνομα`ThreadedCommentsSample.xlsx`, το οποίο περιλαμβάνει ορισμένα σχόλια με νήματα. Βεβαιωθείτε ότι το αρχείο σας περιέχει σχόλια που πρέπει να ακολουθήσετε.

## Εισαγωγή των απαιτούμενων πακέτων

Αρχικά, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων για εργασία με το Aspose.Cells. Ανοίξτε το έργο σας C# και προσθέστε τα ακόλουθα χρησιμοποιώντας οδηγίες στην κορυφή του αρχείου κώδικα:

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

 Ο`Aspose.Cells` Ο χώρος ονομάτων σάς επιτρέπει να έχετε πρόσβαση σε όλες τις κλάσεις και τις μεθόδους που απαιτούνται για τον χειρισμό αρχείων Excel, ενώ`System` απαιτείται για γενική λειτουργικότητα, όπως διαχείριση εξόδου και εξαίρεσης.

## Βήμα 1: Καθορίστε τον Κατάλογο του αρχείου Excel

Το πρώτο βήμα είναι να ορίσετε τη διαδρομή όπου αποθηκεύεται το αρχείο Excel. Αυτή η διαδρομή θα χρησιμοποιηθεί για τον εντοπισμό του αρχείου μέσω προγραμματισμού.

```csharp
// Ορίστε τον κατάλογο του αρχείου Excel
string sourceDir = "Your Document Directory";
```

 Αντικαθιστώ`"C:\\YourDirectory\\"`με την πραγματική διαδρομή προς το αρχείο σας. Αυτό διασφαλίζει ότι το πρόγραμμα γνωρίζει πού να το βρει`ThreadedCommentsSample.xlsx`.

## Βήμα 2: Φορτώστε το βιβλίο εργασίας

 Με το σύνολο του καταλόγου, μπορούμε τώρα να φορτώσουμε το βιβλίο εργασίας του Excel. Αυτό γίνεται με τη δημιουργία ενός νέου`Workbook` αντικείμενο και μεταβίβαση της διαδρομής του αρχείου σε αυτό.

```csharp
// Φορτώστε το βιβλίο εργασίας του Excel
Workbook workbook = new Workbook(sourceDir + "ThreadedCommentsSample.xlsx");
```

Εάν το αρχείο δεν βρεθεί στην καθορισμένη διαδρομή, θα γίνει εξαίρεση, οπότε βεβαιωθείτε ότι η διαδρομή του αρχείου είναι σωστή πριν συνεχίσετε.

## Βήμα 3: Πρόσβαση στο επιθυμητό φύλλο εργασίας

Μόλις φορτωθεί το βιβλίο εργασίας, πρέπει να αποκτήσετε πρόσβαση στο φύλλο εργασίας που περιέχει τα σχόλια με νήματα. Σε αυτό το παράδειγμα, θα ανακτήσουμε το πρώτο φύλλο εργασίας του βιβλίου εργασίας.

```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```

 Εάν τα σχόλιά σας βρίσκονται σε διαφορετικό φύλλο εργασίας, απλώς τροποποιήστε το ευρετήριο ανάλογα. Για παράδειγμα, χρησιμοποιήστε`Worksheets[1]` για το δεύτερο φύλλο εργασίας και ούτω καθεξής.

## Βήμα 4: Ανάκτηση σχολίων με νήματα

Για να ανακτήσετε τα σχόλια με νήματα, θα πρέπει να καθορίσετε το κελί που περιέχει τα σχόλια. Σε αυτή την περίπτωση, εστιάζουμε στο κύτταρο`A1` . Η μέθοδος`GetThreadedComments` χρησιμοποιείται για τη λήψη όλων των σχολίων που σχετίζονται με ένα συγκεκριμένο κελί.

```csharp
// Λάβετε σχόλια με νήματα από το κελί A1
ThreadedCommentCollection threadedComments = worksheet.Comments.GetThreadedComments("A1");
```

Αυτό θα επιστρέψει μια συλλογή σχολίων με νήματα για το κελί A1. Εάν δεν υπάρχουν σχόλια στο καθορισμένο κελί, η συλλογή θα είναι κενή.

## Βήμα 5: Επανάληψη μέσω των σχολίων και εξαγωγή χρόνου δημιουργίας

 Με την ανάκτηση των σχολίων με νήματα, το επόμενο βήμα είναι να επαναλάβετε τη συλλογή και να εξαγάγετε σχετικές λεπτομέρειες, συμπεριλαμβανομένου του χρόνου που δημιουργήθηκε για κάθε σχόλιο. Μπορούμε εύκολα να το πετύχουμε αυτό κάνοντας looping μέσω του`ThreadedCommentCollection`.

```csharp
// Περιηγηθείτε σε κάθε σχόλιο με νήματα και εμφανίστε τις λεπτομέρειες
foreach (ThreadedComment comment in threadedComments)
{
    Console.WriteLine("Comment: " + comment.Notes);
    Console.WriteLine("Author: " + comment.Author.Name);
    Console.WriteLine("Created Time: " + comment.CreatedTime);
}
```

 Αυτός ο κώδικας θα παράγει το περιεχόμενο του σχολίου, το όνομα του συγγραφέα και τη στιγμή που δημιουργήθηκε το σχόλιο. Ο`CreatedTime` Η ιδιότητα επιστρέφει τη χρονική σήμανση όταν δημιουργήθηκε το σχόλιο αρχικά.

## Βήμα 6: Εμφανίστε ένα μήνυμα επιβεβαίωσης

Μετά την επιτυχή ανάγνωση των σχολίων με νήματα και την εμφάνιση των πληροφοριών, είναι πάντα καλή πρακτική να συμπεριλάβετε ένα μήνυμα επιβεβαίωσης στον κώδικά σας. Αυτό βοηθά στην επιβεβαίωση ότι η διαδικασία εκτελέστηκε σωστά.

```csharp
// Μήνυμα επιβεβαίωσης
Console.WriteLine("Successfully retrieved threaded comment created times.");
```

Αυτό το μήνυμα θα εκτυπωθεί στην κονσόλα μόλις ολοκληρωθεί η εκτέλεση του κώδικα, επιβεβαιώνοντας ότι η διαδικασία εκτελέστηκε χωρίς σφάλματα.
