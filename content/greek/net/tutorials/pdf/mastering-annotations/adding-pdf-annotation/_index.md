Πριν από την κωδικοποίηση, βεβαιωθείτε ότι έχετε εισαγάγει τους απαιτούμενους χώρους ονομάτων:

Αυτοί οι χώροι ονομάτων παρέχουν τις κλάσεις και τις μεθόδους που είναι απαραίτητες για χειρισμό και σχολιασμό PDF.

## Βήμα 1: Φορτώστε το έγγραφο PDF σας

Ξεκινήστε φορτώνοντας το έγγραφο PDF στο οποίο θέλετε να προσθέσετε σχολιασμούς PDF.

```csharp
// Καθορίστε τη διαδρομή προς τον κατάλογο των εγγράφων σας.
string dataDir = "YOUR DATA DIRECTORY";
// Φορτώστε το έγγραφο PDF
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");
```

 Αυτό το απόσπασμα κώδικα ορίζει τον κατάλογο για το αρχείο PDF σας και το φορτώνει σε ένα`Document` αντικείμενο, επιτρέποντας περαιτέρω τροποποιήσεις.

## Βήμα 2: Δημιουργήστε έναν σχολιασμό

 Στη συνέχεια, θα δημιουργήσουμε ένα`TextAnnotation`, ιδανικό για την προσθήκη σχολίων ή σημειώσεων.

```csharp
// Δημιουργήστε ένα TextAnnotation
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600))
{
    Title = "Sample Annotation Title",
    Subject = "Sample Subject",
    Contents = "Sample contents for the annotation",
    Open = true,
    Icon = TextIcon.Key
};
```

-  Τοποθεσία και μέγεθος: Το`Rectangle`class ορίζει τη θέση και τις διαστάσεις του σχολιασμού στη σελίδα.
-  Ιδιότητες: Μπορείτε να ορίσετε τον τίτλο, το θέμα και το περιεχόμενο του σχολιασμού. Ο`Open` Η ιδιότητα καθορίζει εάν ο σχολιασμός εμφανίζεται ανοιχτός από προεπιλογή.
-  Εικονίδιο: Το`TextIcon.Key` προσθέτει ένα οπτικό στοιχείο στον σχολιασμό.

## Βήμα 3: Προσαρμόστε την εμφάνιση του σχολιασμού

Βελτιώστε την ορατότητα του σχολιασμού προσαρμόζοντας την εμφάνισή του.

```csharp
// Προσαρμόστε το περίγραμμα του σχολιασμού
Border border = new Border(textAnnotation)
{
    Width = 5,
    Dash = new Dash(1, 1)
};
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

-  Περίγραμμα: Δημιουργία α`Border` αντικείμενο, ρυθμίζοντας το πλάτος και το στυλ του (διακεκομμένη σε αυτήν την περίπτωση) για βελτιωμένη ορατότητα.

## Βήμα 4: Προσθέστε τον σχολιασμό στη σελίδα PDF

Τώρα, ήρθε η ώρα να προσθέσετε τον σχολιασμό στη σελίδα PDF σας.

```csharp
// Προσθέστε τον σχολιασμό στη συλλογή σχολιασμών της σελίδας
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
```

Αυτή η γραμμή προσθέτει τον σχολιασμό που μόλις δημιουργήσατε στην πρώτη σελίδα του PDF, ενσωματώνοντάς τον στο έγγραφο.

## Βήμα 5: Αποθηκεύστε το ενημερωμένο έγγραφο PDF

Τέλος, αποθηκεύστε το έγγραφο για να διατηρήσετε τις αλλαγές σας.

```csharp
// Αποθηκεύστε το ενημερωμένο έγγραφο PDF
dataDir = dataDir + "AddAnnotation_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nAnnotation added successfully.\nFile saved at " + dataDir);
```

Αυτός ο κωδικός αποθηκεύει το τροποποιημένο έγγραφο ως`AddAnnotation_out.pdf`, διατηρώντας το αρχικό αρχείο και επιβεβαιώνοντας την επιτυχή προσθήκη του σχολιασμού.
