## Εισαγωγή

Θέλατε ποτέ να δημιουργήσετε οπτικά εντυπωσιακά PDF μέσω προγραμματισμού; Αν ναι, είστε στο σωστό μέρος! Σε αυτό το σεμινάριο, θα εξερευνήσουμε το Aspose.PDF για .NET, μια ισχυρή βιβλιοθήκη που απλοποιεί τον χειρισμό εγγράφων PDF. Σήμερα, θα επικεντρωθούμε στη δημιουργία ενός γεμάτου ορθογωνίου σε ένα αρχείο PDF. Είτε είστε έμπειρος προγραμματιστής είτε μόλις ξεκινάτε, αυτός ο οδηγός θα σας καθοδηγήσει σε κάθε βήμα με φιλικό και συναρπαστικό τρόπο. Λοιπόν, πάρτε το καπέλο κωδικοποίησης και ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν βουτήξουμε στον κώδικα, βεβαιωθείτε ότι έχετε τα εξής:

1. Visual Studio: Εγκαταστήστε το Visual Studio στον υπολογιστή σας, καθώς είναι ένα εξαιρετικό IDE για ανάπτυξη .NET.
2. Aspose.PDF για .NET: Λήψη και εγκατάσταση της βιβλιοθήκης Aspose.PDF από[εδώ](https://releases.aspose.com/pdf/net/).
3. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε καλύτερα τα αποσπάσματα κώδικα.

## Βήμα 1: Δημιουργήστε ένα νέο έργο

1. Ανοίξτε το Visual Studio και δημιουργήστε ένα νέο έργο εφαρμογής Κονσόλας.
2. Ονομάστε το έργο σας κατάλληλα.

## Βήμα 2: Προσθήκη αναφοράς Aspose.PDF

1. Κάντε δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων.
2. Επιλέξτε Διαχείριση πακέτων NuGet.
3. Αναζητήστε το Aspose.PDF και εγκαταστήστε την πιο πρόσφατη έκδοση.

```csharp
using System.IO;
using System;
using Aspose.Pdf;
```

Τώρα που έχουμε ρυθμίσει τα πάντα, ας βουτήξουμε στον κώδικα!

## Βήμα 3: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Καθορίστε τη διαδρομή όπου θα αποθηκευτεί το PDF σας:

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή στο μηχάνημά σας όπου θέλετε να αποθηκεύσετε το PDF.

## Βήμα 4: Δημιουργήστε μια παρουσία εγγράφου

Αρχικοποιήστε ένα νέο έγγραφο PDF:

```csharp
// Δημιουργία παρουσίας εγγράφου
Document doc = new Document();
```

## Βήμα 5: Προσθέστε μια Σελίδα στο Έγγραφο

Κάθε PDF χρειάζεται τουλάχιστον μία σελίδα. Ας προσθέσουμε ένα:

```csharp
// Προσθήκη σελίδας σε σελίδες συλλογής αρχείου PDF
Page page = doc.Pages.Add();
```

## Βήμα 6: Δημιουργήστε μια παρουσία γραφήματος

 ΕΝΑ`Graph` Το παράδειγμα λειτουργεί ως καμβάς για τη σχεδίαση σχημάτων:

```csharp
// Δημιουργία παρουσίας γραφήματος
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(100.0, 400.0);
```

## Βήμα 7: Προσθέστε το γράφημα στη σελίδα

Επισυνάψτε το γράφημα στη σελίδα:

```csharp
// Προσθήκη αντικειμένου γραφήματος στη συλλογή παραγράφων της παρουσίας σελίδας
page.Paragraphs.Add(graph);
```

## Βήμα 8: Δημιουργήστε μια παρουσία ορθογωνίου

Προσδιορίστε τη θέση και το μέγεθος του ορθογωνίου:

```csharp
// Δημιουργήστε παράδειγμα ορθογώνιου
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 200, 120);
```

## Βήμα 9: Καθορίστε το χρώμα πλήρωσης

Επιλέξτε ένα χρώμα για το ορθογώνιο σας. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε κόκκινο:

```csharp
// Καθορίστε το χρώμα πλήρωσης για το αντικείμενο Graph
rect.GraphInfo.FillColor = Aspose.Pdf.Color.Red;
```

## Βήμα 10: Προσθέστε το Ορθογώνιο στο Γράφημα

Προσθέστε το ορθογώνιο στο γράφημα:

```csharp
// Προσθήκη ορθογώνιου αντικειμένου στη συλλογή σχημάτων του αντικειμένου Graph
graph.Shapes.Add(rect);
```

## Βήμα 11: Αποθηκεύστε το έγγραφο PDF

Τέλος, αποθηκεύστε το έγγραφό σας στον καθορισμένο κατάλογο:

```csharp
dataDir = dataDir + "CreateFilledRectangle_out.pdf";
// Αποθήκευση αρχείου PDF
doc.Save(dataDir);
```

## Βήμα 12: Μήνυμα επιβεβαίωσης

Εκτυπώστε ένα μήνυμα επιβεβαίωσης για να υποδείξετε την επιτυχία:

```csharp
Console.WriteLine("\nFilled rectangle object created successfully.\nFile saved at " + dataDir);
```
