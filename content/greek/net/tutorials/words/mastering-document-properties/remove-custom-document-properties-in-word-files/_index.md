## Φόρτωση του εγγράφου του Word

Το πρώτο βήμα για την τροποποίηση ενός εγγράφου του Word είναι η φόρτωσή του στην εφαρμογή σας. Δείτε πώς μπορείτε να φορτώσετε ένα έγγραφο χρησιμοποιώντας το Aspose.Words για .NET:

### Βήμα 1: Καθορίστε τη διαδρομή αρχείου

Πρέπει να ορίσετε τη διαδρομή αρχείου του εγγράφου του Word. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε το έγγραφο`Properties.docx`.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Properties.docx");
```

 Βεβαιωθείτε ότι έχετε αντικαταστήσει`"YOUR DOCUMENT DIRECTORY"` με τον πραγματικό κατάλογο όπου είναι αποθηκευμένο το έγγραφό σας.

## Πρόσβαση και αφαίρεση των ιδιοτήτων προσαρμοσμένου εγγράφου

Μόλις φορτωθεί το έγγραφο στην εφαρμογή σας, μπορείτε να αποκτήσετε πρόσβαση στις προσαρμοσμένες ιδιότητές του και να τις καταργήσετε. Δείτε πώς να χειριστείτε αυτήν την εργασία:

### Βήμα 2: Ανάκτηση των ιδιοτήτων προσαρμοσμένου εγγράφου

 Αποκτήστε πρόσβαση στις προσαρμοσμένες ιδιότητες του φορτωμένου εγγράφου χρησιμοποιώντας το`CustomDocumentProperties` ιδιοκτησία. Αυτό σας επιτρέπει να διαχειρίζεστε και να τροποποιείτε τις ιδιότητες του εγγράφου μέσω προγραμματισμού.

```csharp
var customProperties = doc.CustomDocumentProperties;
```

### Βήμα 3: Αφαίρεση συγκεκριμένων ιδιοτήτων

 Εάν πρέπει να καταργήσετε μια προσαρμοσμένη ιδιότητα, απλώς καθορίστε το όνομα της ιδιότητας. Για παράδειγμα, ας υποθέσουμε ότι θέλετε να καταργήσετε την ιδιότητα που ονομάζεται`"Authorized Date"`. Εδώ είναι ο κώδικας για αυτό:

```csharp
customProperties.Remove("Authorized Date");
```

 Καλώντας το`Remove` μέθοδο και περνώντας το όνομα του ακινήτου, μπορείτε εύκολα να διαγράψετε τυχόν περιττά ή ξεπερασμένα ακίνητα.

## Αποθήκευση του τροποποιημένου εγγράφου

Μετά την κατάργηση των προσαρμοσμένων ιδιοτήτων, το τελευταίο βήμα είναι να αποθηκεύσετε το τροποποιημένο έγγραφο. Αυτό διασφαλίζει ότι εφαρμόζονται όλες οι αλλαγές, συμπεριλαμβανομένης της κατάργησης προσαρμοσμένων ιδιοτήτων.

### Βήμα 4: Καθορίστε τη διαδρομή αποθήκευσης

Καθορίστε τη διαδρομή στην οποία θέλετε να αποθηκεύσετε το τροποποιημένο έγγραφο. Αυτή είναι η θέση όπου θα αποθηκευτεί το νέο αρχείο Word.

```csharp
string savePath = dataDir + "ModifiedProperties.docx";
```

### Βήμα 5: Αποθηκεύστε το έγγραφο

 Τέλος, χρησιμοποιήστε το`Save` μέθοδος αποθήκευσης του εγγράφου στην καθορισμένη διαδρομή:

```csharp
doc.Save(savePath);
```

Αυτό θα αποθηκεύσει το έγγραφο με τις προσαρμοσμένες ιδιότητες που έχουν αφαιρεθεί, διασφαλίζοντας ότι οι αλλαγές είναι επίμονες.
