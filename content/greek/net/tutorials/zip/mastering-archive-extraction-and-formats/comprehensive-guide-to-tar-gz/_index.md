## Εισαγωγή χώρων ονομάτων

Ξεκινήστε το έργο σας .NET εισάγοντας τους απαραίτητους χώρους ονομάτων για πρόσβαση στις λειτουργίες Aspose.Zip:

```csharp
using System;
using Aspose.Zip.Tar;
```

## Βήμα 1: Ορίστε τον Κατάλογο Εγγράφων σας

Ξεκινήστε ορίζοντας τον κατάλογο όπου βρίσκονται τα αρχεία σας. Αυτό θα αναφέρεται σε όλη τη διαδικασία συμπίεσης.

```csharp
string dataDir = "Your Document Directory";
```

## Βήμα 2: Δημιουργία αρχείου TarGz

Τώρα, ας προχωρήσουμε στη δημιουργία ενός αρχείου TarGz με το Aspose.Zip για .NET ακολουθώντας αυτά τα υποβήματα:

### Βήμα 2.1: Αρχικοποιήστε το TarArchive

 Αρχικά, αρχικοποιήστε ένα`TarArchive` αντικείμενο εργασίας με:

```csharp
using (TarArchive archive = new TarArchive())
{
    // Προσθέστε αρχεία στην αρχειοθέτηση όπως περιγράφεται στα επόμενα βήματα
}
```

### Βήμα 2.2: Προσθήκη καταχωρήσεων

 Στη συνέχεια, προσθέστε τα αρχεία που θέλετε να συμπιέσετε στο αρχείο. Εδώ είναι ένα παράδειγμα με τα αρχεία`"alice29.txt"` και`"lcet10.txt"`:

```csharp
archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");
```

### Βήμα 2.3: Αποθήκευση ως Gzipped Tar

 Τέλος, αποθηκεύστε το δημιουργημένο αρχείο σε μορφή TarGz χρησιμοποιώντας το`SaveGzipped` μέθοδος:

```csharp
archive.SaveGzipped(dataDir + "archive.tar.gz");
```
