Μπορεί επίσης να χρησιμοποιηθεί για τη δημιουργία διαφορετικών εγγράφων Microsoft Word, XML, μορφών OpenDocument και αρχείων TXT.

**Welcome to the GroupDocs.Watermark for Java**

Το GroupDocs.Watermark είναι ένα API για την προσθήκη υδατογραφημάτων σε έγγραφα διαφορετικών μορφών. Παρέχει τις αποτελεσματικές μεθόδους υδατογράφησης που σας επιτρέπουν να προσθέτετε υδατογραφήματα που είναι δύσκολο να αφαιρεθούν αυτόματα από εργαλεία τρίτων. Σας επιτρέπει επίσης να αναζητήσετε και να αφαιρέσετε υδατογραφήματα δημοφιλών τύπων που προστέθηκαν προηγουμένως (συμπεριλαμβανομένων υδατογραφημάτων που προστέθηκαν από εργαλεία τρίτων) σε ένα έγγραφο. Το API παρέχει ένα απλό και εύχρηστο σύνολο μεθόδων για την προσθήκη, αναζήτηση και αφαίρεση υδατογραφημάτων σε υποστηριζόμενες μορφές αρχείων.
